Seaworthy
=========

A ship sailing game.

Assets from Kenney's [Pirate Pack](https://kenney.nl/assets/pirate-pack), [Fish Pack](https://www.kenney.nl/assets/fish-pack)
and [Platformer Art: Requests](https://www.kenney.nl/assets/platformer-art-requests).
